About the Project

A production-ready Natural Language Processing toolkit built entirely on Python's standard library — developed as part of the Advanced Programming course at SRH University.

The Problem

Most Python NLP libraries like NLTK, spaCy, and TextBlob require heavy external dependencies, complex installation, and large model downloads. For many text analysis tasks — especially in educational and lightweight production contexts — this overhead is unnecessary.

Our Solution

The Text Analysis Toolkit provides essential NLP functionality — tokenization, frequency analysis, sentiment scoring, and report generation — with zero external dependencies. Install it and use it immediately.

Design Philosophy

Every class and function is designed for clarity and composability. Use modules independently or combine them through the TextAnalyzer facade. The API mirrors natural language so your code reads like a description of what it does.

Built With

Python 3.9+ Modern type hints, dataclasses, and standard library features
collections.Counter Efficient frequency counting and n-gram aggregation
re (regex) Tokenization, punctuation removal, sentence splitting
argparse CLI argument parsing with help and validation
pathlib Cross-platform file path handling
unittest Comprehensive test suite with 46+ test cases
FastAPI Modern async web framework for this interface
setuptools Packaging and distribution via pyproject.toml
