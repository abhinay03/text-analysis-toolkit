Everything You Need for Text Analysis

Five tightly integrated modules that work together or standalone.

Tokenization

Split text into words and sentences, remove punctuation, filter stopwords, and generate n-grams with full control over preprocessing.

Tokenizer(lower=True, remove_punct=True)

Frequency Analysis

Count words, find the most frequent terms, compute relative frequencies, and filter words above configurable thresholds.

FrequencyAnalyzer(tokens).top_words(10)

Sentiment Scoring

Lexicon-based sentiment analysis with 100+ built-in positive and negative words. Support for custom lexicons and detailed scoring reports.

SentimentAnalyzer().score(tokens)

Report Generation

Generate beautifully formatted plain text and JSON reports with statistics, frequency tables, n-grams, and sentiment summaries.

ReportGenerator(title="Report").generate_text(...)

Unified API

The TextAnalyzer facade runs the entire pipeline — tokenize, analyze frequency, score sentiment, and generate n-grams — in one call.

TextAnalyzer().analyze("Your text here")

CLI Ready

Simple, Clean API

Designed for readability. Works as a library or from the command line.

from text_analysis_toolkit import TextAnalyzer

# Analyze any text in one line
ta = TextAnalyzer(remove_stopwords=True)
result = ta.analyze("I love this amazing product! It is fantastic.")

print(result["statistics"])
# {total_words: 12, unique_words: 11, sentences: 2, ...}

print(result["sentiment"]["sentiment_label"])
# "positive"

print(ta.report_text(result))
# Formatted text report with tables
# Analyze a text file
text-analysis samples/positive_review.txt

# JSON output
text-analysis samples/mixed_article.txt -f json

# Remove stopwords
text-analysis samples/negative_review.txt --no-stopwords

# Analyze inline text
text-analysis -t "I love this amazing product!"

# Save report to file
text-analysis samples/article.txt -o report.txt

# Install locally
pip install -e .
